# Blog-Website-Thing

This is a django-based web application for my blog. It is currently in active development, so the repo is a work in progress.

This project is intended to be used in a linux environment. Project development is used entirely with Docker.

# Installation

1. Clone the repository in the linux environment
2. Ensure Docker is installed

# Run Development environment

Run `docker compose up -d` in the CLI.

# Run Test Suite

Run `docker compose up django-test` in the CLI.
